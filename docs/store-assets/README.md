# Store Assets

Keep Chrome Web Store publishing assets here while preparing the listing.

## Recommended Folder Layout

- `screenshots/` - final localized screenshots for the store listing
- `promo/` - small promo tile and optional marquee tile
- `source/` - editable source files, exported frames, or design files

## Current Chrome Web Store Asset Requirements

Verified against the official Chrome documentation on 2026-07-14:

- Store icon: 128 x 128 px
- Screenshots: at least one 1280 x 800 px screenshot, up to 5 total
- Small promo tile: 440 x 280 px PNG or JPEG
- Marquee promo tile: 1400 x 560 px PNG or JPEG, optional
- Promo video: optional YouTube URL

Before the final upload, re-check:

1. Chrome docs: https://developer.chrome.com/docs/webstore/cws-dashboard-listing
2. Chrome docs: https://developer.chrome.com/docs/webstore/images
3. Chrome Web Store Developer Dashboard validation after dragging files into the listing.
