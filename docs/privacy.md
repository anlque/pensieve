# Memory Bowl Privacy Statement

Last updated: 2026-07-14

Memory Bowl is a local-first Chrome extension and web app for writing, viewing, editing, and releasing personal thoughts.

## Data Stored By The App

Memory Bowl may store:

- thoughts entered by the user
- selected language
- selected wallpaper
- a custom wallpaper image uploaded by the user

This data is stored locally in the user's browser storage or Chrome extension storage.

## How Data Is Used

Stored data is used only to provide Memory Bowl's visible features, such as showing saved thoughts, remembering the selected language, and restoring wallpaper preferences.

## Data Sharing

Memory Bowl does not sell, share, transmit, or disclose user thoughts or preferences to external servers, advertisers, analytics providers, or other third parties.

## Accounts And Tracking

Memory Bowl does not require an account and does not use analytics, advertising trackers, cookies, or profiling.

## Website Content And Browsing Activity

Memory Bowl does not read, collect, or modify the contents of websites the user visits. The `tabs` permission is used only to focus an existing Memory Bowl tab when the extension toolbar icon is clicked.

## Permissions

Memory Bowl uses the `storage` permission to save thoughts and preferences locally.

Memory Bowl uses the `tabs` permission only to find and focus an already-open Memory Bowl extension page instead of opening duplicate tabs.

## Deleting Data

Users can delete individual thoughts, release all thoughts inside the app, reset a custom wallpaper image, or clear the extension's stored data from Chrome's extension/site storage settings.
