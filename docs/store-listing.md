# Chrome Web Store Listing Copy

## Product Name

Memory Bowl

## Short Description

A calm, local-first space to unload, organize, and release your thoughts.

## Category

Recommended primary category: Productivity

Alternative if the dashboard offers a more suitable wellness-focused category: Well-being.

## Detailed Description

Memory Bowl gives you a quiet place to unload thoughts when your mind feels crowded.

Write a thought, catch it with the wand, and let it settle into the bowl. Your thoughts float softly in the water, where you can revisit them, mix them, edit them, or drag them away when you are ready to release them.

Designed as a small calming ritual, Memory Bowl keeps the experience focused and private:

- Add thoughts through a gentle wand animation
- Watch thoughts float in a visual bowl
- Mix the bowl to inspect thoughts up close
- Edit or delete individual thoughts
- Drag thoughts out of the bowl to release them
- Release all thoughts with a calming completion screen
- Choose built-in wallpapers or upload your own local image
- Switch between English and Russian

Memory Bowl stores your thoughts and preferences locally in your browser or Chrome extension storage. It does not require an account, does not send your thoughts to a server, and does not read the web pages you visit.

## Single Purpose Statement

Memory Bowl provides a calm local workspace for writing, viewing, editing, and releasing personal thoughts.

## Permission Justifications

### `storage`

Used to save thoughts, selected language, wallpaper choice, and uploaded custom wallpaper locally in Chrome extension storage. The extension does not transmit this data to external servers.

### `tabs`

Used only when the toolbar icon is clicked to find and focus an already-open Memory Bowl tab instead of opening duplicates. The extension does not read or collect browsing history or page content.

## Remote Code Declaration

No. Memory Bowl does not execute remotely hosted code. The extension is packaged with local JavaScript, CSS, images, and manifest files.

## Data Disclosure Draft

Memory Bowl stores user-generated thoughts and app preferences locally on the user's device using browser storage or Chrome extension storage. This data is used only to provide the app's visible features. It is not sold, shared, transmitted to external servers, used for advertising, or used for tracking.
